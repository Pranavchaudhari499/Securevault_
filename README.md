## Quick Start

### 1. ML Service
```bash
cd ml-service && pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Backend
```bash
cd backend && npm install
cp .env.example .env  # fill in your MongoDB URI
npm run dev
```

### 3. Frontend
```bash
cd frontend && npm install && npm start
```
